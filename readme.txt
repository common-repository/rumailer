== Description ==

RUMAILER – пожалуй, ЛУЧШИЙ на сегодня в рунете сервис email маркетинга с точки зрения вероятности доставки писем, удобства использования, поддержки и, что главное – ПЕРСПЕКТИВЫ развития. Регистрируйтесь, пользуйтесь и оставайтесь с нами!

Возможности:

*   Вновь зарегистрированные пользователи будут автоматически попадать в Ваш подписной лист.
*   Вы всегда будете в курсе Вашего баланса и лимита отправок.
*   Удобный виджет для размещения "Формы подписки".
*   Открытая API даёт возможность Вашим разработчикам встраивать функции в Ваш сайт.
*   Экспорт ваших подсписков в подписные листы Rumailer

Подробные этапы настройки http://rumailer.ru/page/plugins_wordpress/#setting


== Installation ==

1. Переходим в Плагины -> Добавить новый.
2. Нажимаем кнопку «Загрузить плагин».
3. Нажимаем «Выберите файл» и ищем .zip архив с нужным вам плагином на компьютере, после чего жмем кнопку «Установить».
4. После установки плагина активируйте его нажав кнопку «Активировать плагин».

== Frequently Asked Questions ==

= Могу ли я использовать данный плагин без Rumailer? =

Нет. Без сервиса remailer данный плагин не будет работать. Вам необходимо пройти регистрацию на remailer.ru.

= Смогу ли я делать рассылку по пользователям? =

Да. Только по тем пользователям, которые были зарегистрированные и пользователям, которые подписались на обратную форму.

= Могу ли я с помощью этого плагина добавить моих пользователей в подписные листы? =

Да. После настройки плагина у вас будет возможность экспортировать своих пользователей в подписной лист Rumailer
